# VersaTiles Fonts

free font glyphs for map rendering

## Install

```bash
git clone git@github.com:versatiles-org/versatiles-fonts.git
cd versatiles-fonts
npm install
```

## Build

```bash
npm run build
```

- Generates a `dist/fonts.tar.gz` with all glyphs.
- Also generates `dist/{font_family}.tar.gz` for each font family.
